On top of my resources, there is plenty of external resources that are plenty helpful to the community.

## Actions lists

- [sdras/awesome-actions](https://github.com/sdras/awesome-actions/): A list of resources and actions created by [Sarah Drasner](https://twitter.com/sarah_edo) that were community generated into quite the Awesome repo.
- [GitHub Actions Listing on explore-tech.org](https://github-actions.explore-tech.org/): A listing of Actions managed by [David Boyne](https://github.com/boyney123)
- [GitHub Marketplace](https://github.com/marketplace?type=actions): A listing of over 1k actions that were published onto GitHub's marketplace.

## Blogposts

- [GitHub Actions Advent Calendar (Edward Thomson)](https://edwardthomson.com/blog/github_actions_advent_calendar.html): Advent Calendar about GitHub Actions
- [GitHub Actions Advent Calendar (Benjamin Lannon)](https://lannonbr-actions-advent-calendar.netlify.com/): Advent Calendar showcasing individual Actions
- [An Unintentionally Comprehensive Introduction to GitHub Actions CI](https://dev.to/bnb/an-unintentionally-comprehensive-introduction-to-github-actions-ci-blm): A blogpost about Actions by Tierney Cyren on Dev.to

## Tutorials

- [GitHub Actions at Work](https://egghead.io/playlists/github-actions-at-work-ad05f3ff): An Egghead course by Chris Biscardi on setting up things in GitHub Actions.
